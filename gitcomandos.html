Comandos Git

configuração Básica

  • git config --global user.name "[nome da conta do git]"
  • git config --global user.email "[email vinculado a conta do git]"

    • --global indica que as configurações serão utilizadas em todos os repositorio

  • git config -l

    • visualizar todas as condigurações inclusive as globais


    clonando um reposito remoto

    • via HTPPS
    • via SSH

    inicialização das atividades:

    • git fetch --all

      sincroniza o repositorio local com o reposito remoto

    • git pull

      faz o download das atualizações do servidor remoto


    finalização das atividades:

    • git status

      lista as modificações

    • git add .

      adiciona as modificações no servidor local

    • git commit -m "[texto]"

      monta as modificações para envio ao servidor remoto

    • git push

      faz upload das modificações do servidor local para o servidor remoto
